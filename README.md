# ThreadPool_CPP

## 线程池的C++实现

当前结构目录如下：

``` bash
.
├── CMakeLists.txt
├── README.md
├── build
├── include
│   ├── TaskQueue.h
│   └── ThreadPool.h
├── main.cpp
└── src
    ├── TaskQueue.cpp
    └── ThreadPool.cpp
```

- main.cpp：测试线程池的主函数
- TaskQueue.h：任务队列的头文件
- TaskQueue.cpp：任务队列的实现
- ThreadPool.h：线程池的头文件
- ThreadPool.cpp：线程池的实现
